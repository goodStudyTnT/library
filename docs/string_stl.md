# String's STL

## 概要
提供一些常用的字符串操作函数

- `vector<string> split(string &s, string delimiter)`: 将字符串 `s` 按照 `delimiter` 分割，返回分割后的字符串数组。
- `bool isFloat(string &s)`: 判断字符串 `s` 是否为浮点数。
- `std::string string_format(const std::string &format, Args... args)`: 格式化字符串，类似于 `printf` 函数。
